# AntiLife Framework - GUI Edition

## Description
A modern GUI implementation of the AntiLife Framework using Python and CustomTkinter. 
This tool provides a graphical interface for various cybersecurity and network testing utilities.

## Requirements
- Python 3.x
- Standard Linux utilities for network attacks (aircrack-ng, hcitool, etc.) if running those features.
- Windows is supported for Wordlist Generation, Decryption (if tools installed), and basic functions.

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the main application:
```bash
python main.py
```
